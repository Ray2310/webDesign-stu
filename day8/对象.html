Js中关于对象的学习

目标: 知道对象数据类型的特征，具备对象数组 数据渲染页面的能力

对象的定义

Object: 也是一种数据类型。 可以理解为一种数据集合

对象的声明

第一种: let 对象名 = {}
第二种: let 对象名 = new Object()

对象的使用

属性 和 方法组成
let 对象名 = {
属性名: 属性值,
方法名: 函数
}

注意事项:
1. 属性都是成对出现的， 包括属性名 和 值
2. 多个属性之间使用英文分割
3. 属性就是依附在对象上的变量
4. 属性名可以使用"" 或者 '' ，一般情况下省略。

对象的CRUD操作

对象的查找

对象的查可以通过对象.属性 来获取属性的内容。

注意，如果属性中带有字符串 以及 -、 _等 ，查找的时候可以通过
对象名['属性名'] 来获取value

对象的修改

和变量的修改是一样的。 对象.属性 = new值

对象的add

和变量的修改是一样的，但是它是新增属性名 比如一个对象中没有address。我们需要新增， 所以就可以通过
对象.address = value 来实现对象的add

对象中的方法

1. 方法都是成对出现的， 包括方法名 和 函数
2. 多个属性之间使用英文分割
3. 方法就是依附在对象中的函数
4. 属性名可以使用"" 或者 '' ，一般情况下省略。

fun: function (){
console.log("这是对象的方法， 其中的方法名为sum")
}

动态添加属性和方法

扩展：也可以动态为对象添加方法，动态添加与直接定义是一样的，只是语法上更灵活。

注：无论是属性或是方法，同一个对象中出现名称一样的，后面的会覆盖前面的。

null的类型也是object :

null 也是 JavaScript 中数据类型的一种，通常只用它来表示不存在的对象。使用 typeof 检测类型它的类型时，结果为 `object`。

内置对象

回想一下我们曾经使用过的 `console.log`，`console`其实就是 JavaScript 中内置的对象，该对象中存在一个方法叫 `log`，然后调用 `log` 这个方法，即 `console.log()`。 除了 `console` 对象外，JavaScritp 还有其它的内置的对象

`Math` 是 JavaScript 中内置的对象，称为数学对象，这个对象下即包含了属性，也包含了许多的方法。

such as:
Math.random() // 0 ~ 1 之间的随机数, 包含 0 不包含 1
Math.ceil(3.4) // 数字向上取整 舍弃小数部分，整数部分加1
Math.max(10, 21, 7, 24, 13) //取最大的value
