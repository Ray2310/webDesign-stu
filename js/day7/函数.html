函数的学习

函数的基本使用

函数的声明: function函数名(){
函数体
}

//求两个数的和
function add( a, b){ //如果声明了之后， 但是并没有赋值， 那么就是undefined
let res = a + b
return res
}

函数的参数&返回值

注意:
函数的参数最好赋值一个默认值， 如果不赋值的话， 那么一旦没有传参数或者传的参数不符合， 那么就会造成程序报错， 最终导致程序奔溃。
如果不传参， 那么参数的默认类型是undefined类型。 所以我们一般将函数形参默认写成0 NaN

函数的返回值

如果没有return 默认返回undefined类型;
return 后面的数据不能换行写， 因为js默认换行就是语句结束

函数的作用域

两个相同的函数， 后面的函数会覆盖前面的函数；
在js中， 实参和形参的个数可以不一致

全局作用域 和 局部作用域
全局作用域可以在另一个js文件中写，也可以写在一个js文件的里面，但是是在函数等的外面。
局部作用域就是一般在函数或者循环语句中的/还可以是定义在代码块中的。

变量的访问原则

如果变量内部中的一个变量没有声明却直接赋值， 那么这个变量也会被当成全局变量来看。但是强烈不推荐

1. 只要是代码，就至少有一个作用域
2. 在能够访问到的情况下，先局部，局部没有找到再找全局

匿名函数

function(){} : 这样就是一个匿名函数

匿名函数的实现

1. 函数表达式
2. 立即执行函数

1. 函数表达式， 将函数赋值给一个变量， 并且通过一个变量名称进行调用。我们将之称为函数表达式
let fn = function (){ //函数体
}

匿名函数必须先声明再调用。多个立即执行函数之间必须用;隔开

2. 立即执行函数: 说的就是不需要调用， 写完就会立即执行。
使用场景: 避免全局变量的污染。
方式一: (function () {
//函数体
})();
方式二:
(function () {
//函数体
}());

转换为boolean类型

对于字符串来说除了"" 其余转换为Boolean都为true
对于Number类型来说， 除了0 之外 其余的转换为Boolean都为真。
undefined 、 null 、false7NaN 转换为Boolean类型都是false 剩下的都为true
