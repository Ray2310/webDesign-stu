获取DOM元素

let div = document.querySelector('div')
console.log(div)

万事万物皆为对象

通过CSS选择器的方式获取DOM对象

获取某个元素就是利用js获取页面中的元素
1. 获取匹配的第一个元素 document.querySelector('css选择器')

  • 1.1 获取匹配的第一个元素 document.querySelector('.calssName(类名)')
  • 1.2 获取匹配的第一个元素 document.querySelector('#id内容(id号)')
  • 1.3 获取匹配的第一个元素 document.querySelector('ul li:first-child')
2. 获取匹配的多个元素 document.querySeWlectorAll('css选择器') 得到的结果为一个对象数组。
通过这个得到的内容是一个伪数组，有长度和索引号的数组，但是没有pop和push等数组的方法。想要得到里面的内容，就需要遍历(for)的方式获取
3. 根据id获取一个元素 document.getElementById('nav')
4. 根据标签获取一类元素，获取页面所有的div document.getElementsByTagName('div')
5. 根据类名获取元素 获取页面中所有类名为w的 document.getElementsByClassName('W')

操作元素内容

DOM对象是根据标签生成的，所以操作标签就是操作dom对象
我们可以通过对象.innerText属性 来操作标签中的内容
或者是通过对象.innerHTML属性(相对于上面来说唯一的区别就是可以解析标签) 来操作标签中的内容

操作元素属性

1. 修改元素属性，就是操作标签的大小或者标签的颜色、背景色等等 也是通过对象.属性来实现属性的修改
2. 修改style格式中的相关信息 可以通过div.style.color = 'red' 为例来进行修改。如果遇到样式名有-的， 可以使用小驼峰命名法来实现

2. 通过类名className操作CSS

适合修改样式比较多的情况。直接通过style属性修改比较繁琐。可以借助css类名的形式
元素.className = 'active' //active是一个css类名

通过classList操作控制CSS

为了解决className容易覆盖以前的类名， 我们可以通过classList的方式追加和删除类名
语法: 元素.classList.add('类名') //追加
元素.classList.remove('类名') //直接删除
元素.classList.toggle('类名') //切换一个类，有就删除，没有就添加

操作表单元素属性

表单很多情况需要进行修改属性， 比如点击眼睛查看密码，本质上就是将表单类型(passwd)转换为文本框(text)
表单.value = '用户名'
表单.type = 'password'

console.log(username.value) //username.innerHTML得不到表单的内容，只能通过username.value

表单中的某些属性添加了就有效果，没有添加就不会有效果
这种情况一律使用boolean类型来代表该属性
