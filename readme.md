# day7 函数 数组相关stu
1. 其中的匿名函数的两种实现方式需要牢记。
2. 如果变量内部中的一个变量没有声明却直接赋值， 那么这个变量也会被当成全局变量来看。但是强烈不推荐
3. 函数的参数最好赋值一个默认值， 如果不赋值的话， 那么一旦没有传参数或者传的参数不符合， 那么就会造成程序报错， 最终导致程序奔溃。<br> 如果不传参， 那么参数的默认类型是undefined类型。 所以我们一般将函数形参默认写成0 NaN
